<p align="center"><code>git-dld</code> download files or folder from GitHub.</p>

![Demo](https://i.imgur.com/dARBpUw.gif)

## Installation

```bash
npm install -g git-dld
# or you may use yarn
yarn global add git-dld
```

## Usage

```bash
git dld [url] --token [github-access-token]
```
